# Trabalho Enfermagem Mãozinha

## Descrição:
Projeto colaborativo com a turma de Enfermagem, com foco na importância da higienização das mãos. Este foi o site mais completo que desenvolvi até hoje, com interatividade, responsividade e uso avançado de HTML, CSS e JavaScript.

## Objetivo do Projeto:  
- Desenvolver um site educativo sobre a importância da higienização das mãos.  
- Criar interatividade para engajar o usuário e transmitir a mensagem de forma clara.  
- Garantir responsividade para dispositivos móveis e desktop.

## Tecnologias:
- HTML  
- CSS  
- JavaScript

## Funcionalidades
- Layout responsivo
- Estrutura de site institucional
- Seções informativas sobre serviços e contato

## O que foi aprendido
- Organização de um projeto web
- Estruturação de páginas com HTML semântico
- Estilização e responsividade com CSS
- Noções iniciais de colaboração em projetos
